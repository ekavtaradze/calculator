# Simple Calculator

####DESCRIPTION

Calculator program for basic Mathematical equations.

##### Supported Features
- Command Line input
- Addition(+), Subtraction(-), Multiplication(*), Division(/) on all integers and decimals
- ??? 

##### Examples
- "1+2" >>> 3
- "4*10/2" >>> 20
- "-5+-8--11*2" >>> 9
- "-.32   /.5" >>> -0.64
- "2+-+-4" >>> Syntax Error
- "89 + word" >>> Invalid Input
- one with nested


##### Utilizing
 - Maven
 - Java JDK 1.8
 - JUnit
